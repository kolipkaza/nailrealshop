# 💅 Nail_Researcher - Quick Start Guide

## 🚀 วิธีใช้งาน Nail_Researcher

### 📝 สร้าง research task:

```
ช่วย research เกี่ยวกับ [TOPIC]
```

**ตัวอย่าง:**
- "research modern nail salon website themes"
- "find 2026 nail color trends"
- "search nail art portfolio designs"

---

## 🎯 สิ่งที่ Nail_Researcher ทำได้:

### 1. **Website Research**
- ค้นหา theme ร้านทำเล็บที่สวยงาม
- ศึกษา fashion website designs
- วิเคราะห์ UX/UI ของ beauty websites

### 2. **Color Trends**
- ตามหา trend สีเล็บปี 2026
- สร้าง color palette สำหรับ nail salon
- รวบรวม seasonal color trends

### 3. **Inspiration**
- หา nail salon websites ที่ประสบความสำเร็จ
- ศึกษา gallery layouts
- ดู booking system designs

## ⚡ การทำงาน

1. **Spawn agent** → รับ task
2. **Search** → DuckDuckGo search
3. **Compile** → รวบรวมข้อมูล
4. **Save** → บันทึกเป็น .md file
5. **Announce** → แจ้งผลลัพธ์
