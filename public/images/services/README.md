# 📸 วิธีใส่รูปบริการ

## วิธีใช้

1. ดู **Service ID** จากหน้าเว็บ (แสดงบน card)
2. ตั้งชื่อไฟล์รูปให้ตรงกับ ID เช่น:
   - `1.jpg` → บริการ "แมนิคิวคลาสสิก"
   - `2.jpg` → บริการ "แพดิคิวคลาสสิก"
   - `3.jpg` → บริการ "เจลแมนิคิว"
   - และอื่นๆ...

## รูปแบบไฟล์ที่รองรับ
- `.jpg` (แนะนำ)
- `.png`
- `.webp`

## ขนาดแนะนำ
- **Card**: 400x300 px (จะ crop ให้พอดี)
- **Detail**: 600x400 px หรือใหญ่กว่า

## หมายเหตุ
- ถ้าไม่มีรูป จะแสดงไอคอน + ID แทน
- ใส่รูปแล้ว refresh หน้าเว็บ รูปจะขึ้นทันที
